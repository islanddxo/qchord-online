
# QChord Online

A browser-based virtual instrument inspired by the QChord, allowing users to play chords, strum melodies, and trigger rhythms in real time using keyboard or mouse.

## Overview

QChord Online recreates a hardware-style instrument experience in the browser.
Users can select chords, strum across a responsive strumplate, and layer rhythms and voices with real-time audio feedback.

---

## Key Features

* **Chord system** — 10-column layout with major, minor, and 7th chords
* **Strumplate** — 10 interactive sections with mouse, drag, and keyboard input
* **Voices** — Guitar, harp, organ, and synth
* **Rhythms** — Arp, hiphop, rock, and pop
* **Auto chord** — Sustained chords with dynamic switching
* **Controls panel** — Keyboard shortcuts via pull-out menu
* **Responsive scaling** — Entire instrument scales proportionally
